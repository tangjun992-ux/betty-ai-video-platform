AI 短视频自动生成平台

FastAPI + Celery + Redis — 多模型智能路由架构

核心服务

  • • FastAPI 网关 — 异步 API，自动 OpenAPI 文档
  • • Celery Worker — 多队列（image / fast_video / pro_video）
  • • 智能路由引擎 — Prompt 分析 → 模型评分决策
  • • WebSocket — 实时推送任务进度

接入模型（可扩展）

  • • OpenAI GPT-5.4 Image 2 — 高质量图像
  • • ByteDance Seedance 2.0 Fast — 快速短视频
  • • 可灵 Kling Video v3 Pro — 电影级长视频
  • • 适配器模式 — 新模型只需新增一个 Adapter 类

基础设施

  • • PostgreSQL — 用户/任务/账单数据持久化
  • • Redis — 任务队列 / 会话 / 限流 / 缓存
  • • S3 / OSS — 媒体文件存储，CDN 加速分发
  • • Docker + K8s — 容器化，水平扩展

分阶段实施

  • • Phase 1 (2-3周): MVP — 单模型流程打通
  • • Phase 2 (2周): 智能路由 + 多模型
  • • Phase 3 (2周): 积分计费 + 商业化
  • • Phase 4 (持续): 更多模型 + 社区功能
